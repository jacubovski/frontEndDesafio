>### Consiste em criar uma Tela de para CRUD de Cidades e Estados, consumindo a API, também do desafio.

# Requisitos para Instalação
>#### 1. NodeJS 10x+
>#### 2. NPM 6x+

# Instalaçao
### Dependências 
> $ npm install
### Estartando a apicação para desenvolvimento
> $ npm run serve
### Build para Produção
> $ npm run build
### Parando a aplicação
> $ Ctrl + c
